# Manufacturing Line Productivity Analysis

## Project Overview

This project analyzes manufacturing line productivity to identify the primary causes of machine downtime and determine whether downtime is driven mainly by machine-related issues or operator-related factors. The objective is to provide data-driven recommendations that improve production efficiency and reduce operational losses.

---

## Business Problem

A manufacturing company has been experiencing frequent machine downtime, leading to reduced productivity and increased operational costs. Management wants to understand:

* Whether machine downtime is primarily caused by equipment failure or operator-related factors.
* Which factors contribute most to production interruptions.
* What corrective actions can be implemented to improve manufacturing efficiency.

---

## Project Objectives

* Analyze downtime caused by machine failures.
* Evaluate downtime associated with operator performance.
* Identify the major factors contributing to machine downtime.
* Provide actionable recommendations to improve production efficiency.

---

## Dataset

The analysis was performed using the Manufacturing Line Productivity dataset, which contains information on production activities, operators, machine performance, downtime events, and productivity metrics.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning and Transformation
* Data Visualization (Excel Dashboard)

---

## Data Cleaning

Before analysis, the dataset was cleaned to improve data quality and ensure accurate results. The following tasks were performed:

* Corrected spelling inconsistencies.
* Standardized column names.
* Formatted data types appropriately.
* Checked for inconsistent values and cleaned the dataset for analysis.

---

## Analysis Process

The cleaned dataset was analyzed using Excel Pivot Tables to identify patterns and measure production performance.

The analysis included:

* Calculating operator efficiency to assess the impact of human factors on machine downtime.
* Measuring the percentage contribution of each downtime category.
* Comparing machine-related and operator-related downtime.
* Designing an interactive dashboard to communicate key insights effectively.

---

## Key Findings

* Operator efficiency was calculated at **64%**, indicating that operator performance contributed significantly to production downtime.
* Downtime was influenced by both machine-related issues and human factors rather than a single cause.
* The analysis identified the most frequent downtime categories, allowing management to prioritize improvement efforts.
* The dashboard provided a clear visual summary of downtime trends and operational performance.

---

## Dashboard

The dashboard presents:

* Downtime distribution by cause.
* Operator efficiency.
* Major downtime contributors.
* Production performance summary.
* Key performance indicators (KPIs).

---

## Recommendations

1. Conduct machine setup and adjustment training for all operators to reduce operational errors.

2. Provide targeted training on batch changeover procedures to minimize downtime during production transitions.

3. Implement a preventive maintenance schedule and routine machine inspections to reduce unexpected equipment failures.

4. Monitor operator performance continuously using production KPIs and provide refresher training where necessary.

5. Establish standardized operating procedures (SOPs) to ensure consistency in machine operation across all production shifts.

---

## Business Impact

The insights from this analysis enable management to distinguish between equipment-related and operator-related downtime, prioritize maintenance activities, improve workforce performance, and increase overall manufacturing productivity.
